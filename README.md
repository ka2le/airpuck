# Airpuck

Space-themed mobile air hockey built with Phaser + TypeScript + Vite.

## Features
- Two-player local play on one phone/tablet
- Two simultaneous touches tracked
- Nearest available paddle gets grabbed by a touch
- No third grab while two touches are active
- Fast arcade puck with light drag
- Wall bounces reduce speed somewhat
- Paddle movement influences puck rebound speed and angle
- Goals and scoring
- Tap score to open restart menu
- Fullscreen-friendly and landscape-oriented play

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
